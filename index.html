My Works

Stealth Crawl

Solo Project/3 Months

Project Stealth Crawl is an immersive sim dungeon crawler. The goal of this project was for me to create a variety of different versatile systems

My Contributions

  • Team system that allows for AI to have varying alliances. This allows for NPCs to fight each other.
  • AI sound detection. Each sound effect in the game has a varying noise value attached to it. NPCs that hear a sound will go and investigate it.
  • Versatile item system that allows for the rapid creation of new items.
